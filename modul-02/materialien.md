# Modul 2 – Materialien
## Sicher unterwegs – Passwörter, Einstellungen, Datenschutz

### Handout: Passwort-Strategie

**Die Satz-Methode:**
1. Denken Sie sich einen persönlichen Satz aus
2. Nehmen Sie die Anfangsbuchstaben + Zahlen + Satzzeichen
3. Fertig!

| Satz | Passwort |
|------|----------|
| Mein Hund Bello frisst 2 Dosen am Tag! | MHBf2DaT! |
| Ich liebe Bamberg mit seinen 9 Brücken. | IlBms9B. |
| Unser Garten hat 12 Rosenbüsche und 1 Apfelbaum! | UGh12Ru1A! |

**MERKEN:** Das Passwort steht NUR in Ihrem Kopf – nie auf einem Zettel am Computer!

### Checkliste: Smartphone-Sicherheit

- [ ] **Bildschirmsperre eingerichtet** (PIN, Muster oder Fingerabdruck)
- [ ] **WLAN-Passwort geändert** (nicht das Standard-Passwort des Routers)
- [ ] **App-Berechtigungen geprüft** (Kamera, Mikrofon, Standort, Kontakte)
- [ ] **Automatische Updates aktiviert** (Einstellungen → Software-Update)
- [ ] **Unbekannte Apps-Installation blockiert** (Einstellungen → Sicherheit)

### Schritt-für-Schritt: Bildschirmsperre einrichten

**Android:**
1. Einstellungen → Sperrbildschirm
2. "Sperrbildschirmtyp" auswählen
3. PIN wählen (mindestens 6 Ziffern)
4. PIN zweimal eingeben → Fertig!

**iPhone:**
1. Einstellungen → Face ID & Code (oder Touch ID & Code)
2. "Code aktivieren" tippen
3. 6-stelligen Code eingeben
4. Code bestätigen → Fertig!

### Schritt-für-Schritt: App-Berechtigungen prüfen

**Android:**
1. Einstellungen → Apps
2. App auswählen (z.B. "Taschenlampe")
3. "Berechtigungen" tippen
4. Prüfen: Braucht diese App Kamera? Kontakte? Standort?
5. Nicht benötigte Berechtigungen abschalten

**iPhone:**
1. Einstellungen → Datenschutz & Sicherheit
2. Kategorie wählen (z.B. "Kamera")
3. Sehen Sie, welche Apps Zugriff haben
4. Nicht benötigte abschalten
