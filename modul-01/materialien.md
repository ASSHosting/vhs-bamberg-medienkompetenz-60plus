# Modul 1 – Materialien
## Ankommen in der digitalen Welt – Mein Gerät und ich

### Handout: Grundbegriffe

| Begriff | Was ist das? | Wo finde ich das? |
|---------|-------------|-------------------|
| **App** | Ein Programm auf Ihrem Smartphone | Auf dem Homescreen oder im App Store |
| **WLAN** | Drahtloses Internet | Einstellungen → WLAN |
| **Touchscreen** | Der Bildschirm reagiert auf Berührung | Tippen = Klicken, Wischen = Blättern |
| **Homescreen** | Ihre Startseite | Drücken Sie den Home-Knopf |
| **App Store / Play Store** | Der "Laden" für neue Apps | Buntes Dreieck (Android) oder blaues A (Apple) |

### Schritt-für-Schritt: App Store öffnen

**Android:**
1. Suchen Sie auf Ihrem Homescreen das Symbol "Play Store" (buntes Dreieck)
2. Tippen Sie darauf
3. Oben finden Sie ein Suchfeld – tippen Sie hinein
4. Geben Sie "Wetter" ein
5. Schauen Sie sich die Ergebnisse an

**iPhone/iPad:**
1. Suchen Sie auf Ihrem Homescreen das Symbol "App Store" (blaues A)
2. Tippen Sie darauf
3. Unten tippen Sie auf "Suchen"
4. Geben Sie "Wetter" ein
5. Schauen Sie sich die Ergebnisse an
